🐍 Python 제어문

프로그램의 흐름을 제어한다

조건에 따라 다르게 동작하고, 반복적으로 작업을 수행하는 방법을 배워보자!

🤔 왜 제어문을 사용해야 할까?

  • 기본 실행: 프로그램은 위에서 아래로 순차적으로 실행
  • 실제 필요: 조건에 따라 다르게 동작해야 하는 경우가 많음
  • 반복 작업: 같은 작업을 여러 번 수행해야 할 때
  • 유연한 코드: 다양한 상황에 맞는 프로그램 작성 가능

📋 제어문의 종류

1️⃣ 조건문

특정 조건에 따라 명령을 선택적으로 실행

if, elif, else

2️⃣ 반복문

특정 명령을 여러 번 반복 실행

for, while

3️⃣ 분기문

반복 흐름을 중단하거나 건너뜀

break, continue

🔍 비교 연산자

연산자 의미 예시 (a=3, b=5) 결과
==같다a == bFalse
!=같지 않다a != bTrue
>크다a > bFalse
<작다a < bTrue
>=크거나 같다a >= bFalse
<=작거나 같다a <= bTrue

🧠 논리 연산자

and

두 조건이 모두 참일 때만 참

True and True → True
True and False → False

or

둘 중 하나라도 참이면 참

True or False → True
False or False → False

not

조건의 참/거짓을 반대로

not True → False
not False → True

🎯 조건문 (if문)

if문이란? 조건이 참(True)일 때만 특정 코드를 실행하도록 하는 제어문입니다.
조건에 따라 프로그램의 흐름을 분기할 수 있습니다.

score = 85
if score >= 90:
    print("A학점")
elif score >= 80:
    print("B학점")
else:
    print("C학점")
출력: B학점
  • if: 첫 번째 조건을 검사
  • elif: 위 조건이 거짓이면 다음 조건 검사
  • else: 모든 조건이 거짓이면 실행

🔄 while문

조건이 참인 동안 반복 실행

i = 1
while i <= 5:
    print(i)
    i += 1
출력:
1
2
3
4
5

🔁 for문

시퀀스의 요소를 하나씩 꺼내며 반복

numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num)
출력:
1
2
3
4
5

🚦 분기문 (break, continue)

break - 반복 종료

for i in range(1, 10):
    if i == 5:
        break
    print(i)
출력: 1, 2, 3, 4

continue - 건너뛰기

for i in range(1, 6):
    if i == 3:
        continue
    print(i)
출력: 1, 2, 4, 5

📝 정리

🎯 제어문으로 할 수 있는 것들

  • 조건문: 상황에 따라 다른 동작 실행
  • 반복문: 같은 작업을 효율적으로 반복
  • 분기문: 반복의 흐름을 세밀하게 제어

프로그램의 흐름을 자유자재로 조절하여
복잡한 문제도 단계별로 해결할 수 있다! 🚀
